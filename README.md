## Chatterbox is a cross-platform messaging app built with React Native.

### Recent Update
<ul>
  <li>Smoother screen transitions</li>
  <li>Updated graphics</li>
  <li>New color scheme</li>
  <li>Direct message and group messaging capability</li>
  <li>Collapsable timestamps</li>
  <li>Message deletion</li>
</ul>

### Future Updates
<ul>
  <li>Message editing capability</li>
  <li>File uploading and sharing</li>
  <li>Group message invites</li>
  <li>Push notifications</li>
  <li>Message status (unread or read messages)</li>
  <li>User status (online, offline, or typing)</li>
</ul>
